Question 1 Easy
Which of the following statements about money-weighted and time-weighted returns is most accurate?
A Time-weighted returns are preferred for evaluating portfolio manager performance because they are not affected by the timing of client cash flows.
B Money-weighted returns are always higher than time-weighted returns when additional funds are invested.
C Both methods will always produce identical results for the same portfolio.

‚úì Correct Answer: A

Explanation: Time-weighted returns are the preferred method for evaluating portfolio manager performance because they eliminate the effects of cash flows that are outside the manager's control. This allows for fair comparisons between different managers.

Why B is incorrect: Money-weighted returns can be higher or lower than time-weighted returns depending on the timing of cash flows relative to performance. If more funds are invested before strong performance, money-weighted return will be higher; if invested before weak performance, it will be lower.

Why C is incorrect: The two methods produce identical results only when there are no interim cash flows or when all interim cash flows occur at the end of the measurement period.

A 14.9%
B 15.0%
C 16.0%

‚úì Correct Answer: A (14.9%)

Calculation:

Year 1 Return: 10%
Year 2 Return: 20%

Time-Weighted Return = [(1.10) √ó (1.20)]^(1/2) - 1

Key Insight: The time-weighted return (14.9%) is lower than the money-weighted return (16.0%) because it doesn't give extra weight to Year 2's higher return when more capital was invested. It treats each year equally.

Question 4 Easy
Which return measure is most similar to calculating the internal rate of return (IRR)?
A Money-weighted return
B Time-weighted return
C Both are equally similar to IRR

‚úì Correct Answer: A

Explanation: The money-weighted return is calculated using the same methodology as the internal rate of return (IRR). Both solve for the discount rate that makes the present value of all cash flows equal to zero.

The time-weighted return, on the other hand, is calculated using the geometric mean of subperiod returns and does not use the IRR methodology.

Question 5 Medium
A portfolio manager achieves returns of -20%, +30%, and +10% over three consecutive years. An investor contributes $100,000 at the start of Year 1, adds $200,000 at the start of Year 2, and withdraws $50,000 at the start of Year 3. If the money-weighted return exceeds the time-weighted return, this most likely indicates:
A The investor demonstrated favorable timing by adding more capital before strong performance periods.
B The portfolio manager's skill exceeded market expectations.
C The investor demonstrated unfavorable timing by adding capital before weak performance.

‚úì Correct Answer: A

Explanation: When the money-weighted return exceeds the time-weighted return, it indicates that the investor had favorable timing. The investor had more capital invested during periods of strong performance.

In this case:

  • Only $100,000 was exposed to the -20% return in Year 1
  • A much larger amount ($200,000+) was invested during Years 2 and 3 when returns were positive (+30% and +10%)

The time-weighted return treats each period equally, while the money-weighted return gives more weight to periods when more capital was invested, resulting in a higher overall return.

Note: Option B is incorrect because the comparison doesn't tell us anything about the manager's skill relative to market expectations - it only reveals the impact of cash flow timing.

Question 7 Medium
A fund manager reports the following quarterly returns: +8%, -4%, +12%, and +6%. An investor who added funds only at the beginning of the measurement period would find that their:
A Money-weighted return equals their time-weighted return.
B Money-weighted return exceeds their time-weighted return.
C Time-weighted return exceeds their money-weighted return.

‚úì Correct Answer: A

Explanation: When there are no interim cash flows (the investor only invests at the beginning and liquidates at the end), the money-weighted return and time-weighted return will be identical.

Both methods would calculate:

Return = (1.08 √ó 0.96 √ó 1.12 √ó 1.06) - 1 = 22.27% (annual equivalent)

Key Point: The two measures only diverge when there are interim cash flows (additions or withdrawals) during the measurement period. Without interim cash flows, both methods measure the same thing - the compound growth rate of the initial investment.

Question 8 Easy
According to the Global Investment Performance Standards (GIPS), investment firms must present performance using:
A Money-weighted returns only.
B Time-weighted returns only.
C Either money-weighted or time-weighted returns at the firm's discretion.

‚úì Correct Answer: B

Explanation: GIPS requires investment firms to calculate and present time-weighted returns. This requirement ensures fair comparison across different investment managers and funds.

Rationale: Time-weighted returns are not affected by client-directed cash flows (which are outside the manager's control), making them the most appropriate measure for evaluating and comparing manager performance.

While firms may also calculate money-weighted returns for client reporting purposes, GIPS specifically requires time-weighted returns for performance presentation standards.

Question 10 Medium
Two investors, Amit and Priya, invest in the same mutual fund. Amit invests ‚Çπ5,00,000 at the start and adds ‚Çπ2,00,000 after one year. Priya invests ‚Çπ7,00,000 at the start with no additional contributions. After two years, both investors' holdings have the same market value. Which statement is most accurate?
A Amit and Priya will have the same money-weighted return and time-weighted return.
B Amit and Priya will have the same time-weighted return but different money-weighted returns.
C Amit and Priya will have different time-weighted returns and different money-weighted returns.

‚úì Correct Answer: B

Explanation: Both investors are in the same mutual fund, which means the fund's performance (time-weighted return) is identical for both.

Time-Weighted Return: Will be the same for both because it measures the fund's performance independent of cash flows. Both experienced the same sequence of returns.

Money-Weighted Return: Will be different because:

  • Amit made an additional ‚Çπ2,00,000 contribution after one year
  • Priya made no additional contributions
  • Their cash flow patterns differ, leading to different money-weighted returns

Key Insight: This question demonstrates why time-weighted returns are preferred for comparing manager performance - all investors in the same fund have the same time-weighted return, but different money-weighted returns based on their individual cash flow timing.

Question 11 Hard
A portfolio manager's fund generates returns of +15%, -8%, +20%, and +12% over four years. An investor contributes $50,000 at the start of Year 1, withdraws $20,000 at the end of Year 2, and adds $30,000 at the end of Year 3. If the final portfolio value is $110,000, the difference between the time-weighted and money-weighted returns most likely:
A Reflects the investor's skill in market timing.
B Indicates superior portfolio manager performance.
C Results from the interaction between cash flow timing and performance, but doesn't indicate either timing skill or manager performance.

‚úì Correct Answer: C

Explanation: The difference between money-weighted and time-weighted returns results from the interaction between the timing of cash flows and the performance during those periods. However, this difference alone doesn't indicate:

Why it doesn't indicate timing skill (Option A is wrong):

  • The investor may have made contributions/withdrawals for personal reasons unrelated to market timing
  • Cash flows could be driven by life events, not market predictions
  • A single period doesn't establish a pattern of timing skill

Why it doesn't indicate manager performance (Option B is wrong):

  • Manager performance is best measured by time-weighted return alone
  • The difference between the two measures tells us about cash flow effects, not manager skill
  • Comparing to a benchmark would be needed to assess manager performance

The Key Point: The difference between the two measures is a mathematical result of when cash flowed in and out relative to performance. It's descriptive, not evaluative.

Question 12 Medium
An analyst is comparing three mutual funds. Fund A reports a money-weighted return of 18% and time-weighted return of 15%. Fund B reports a money-weighted return of 14% and time-weighted return of 17%. Fund C reports both measures as 16%. Based on this information alone, which fund most likely had the best manager performance?
A Fund A, because it has the highest money-weighted return.
B Fund B, because it has the highest time-weighted return.
C Fund C, because its returns are consistent across both measures.

‚úì Correct Answer: B

Explanation: When comparing manager performance across different funds, use the time-weighted return because it eliminates the effects of cash flows and focuses purely on investment performance.

Analysis of each fund:

  • Fund A: MWR (18%) > TWR (15%) - Investors had favorable timing, adding money before strong performance
  • Fund B: TWR (17%) > MWR (14%) - Best manager performance, but investors had poor timing
  • Fund C: MWR = TWR (16%) - Either no interim cash flows or cash flows occurred at neutral times

Key Insight: Fund B's manager delivered the best performance (17% time-weighted return), even though the fund's investors didn't capture this performance fully due to unfortunate cash flow timing (resulting in only 14% money-weighted return).

Why A is wrong: The high money-weighted return in Fund A reflects investor timing luck, not manager skill.

Why C is wrong: Equal measures don't indicate superior performance; they just indicate minimal cash flow effects. The actual performance (16%) is lower than Fund B's 17%.
