Calculate and Interpret Annualized Return Measures and Continuously Compounded Returns

Learning Outcome Statement

LOS: Calculate and interpret annualized return measures and continuously compounded returns, and describe their appropriate uses.

This LOS requires candidates to understand the mathematical conversion of returns across different time periods, recognize the fundamental differences between discrete and continuous compounding, and apply these concepts appropriately in investment analysis.

1. The Need for Return Standardization

Investment returns are realized over varying time horizons. To facilitate meaningful comparison across different securities and investment strategies, returns must be standardized to a common temporal basis. The convention in finance is to express returns on an annualized basis.

Core Principle

Annualization is the process of converting a return earned over any period (shorter or longer than one year) to an equivalent annual rate of return through the application of compound interest principles.

2. Non-Annual Compounding Framework

When interest compounds more frequently than annually, the present value formula must be adjusted to account for the periodic interest rate and the corresponding number of compounding periods.

Present Value with Non-Annual Compounding

PV = FVN × (1 + Rs/m)-mN

Where:

  • PV = Present value
  • FVN = Future value at time N
  • Rs = Quoted annual interest rate (stated annual rate)
  • m = Number of compounding periods per year
  • N = Number of years

Key Insight: The periodic interest rate is Rs/m, and the total number of compounding periods is mN.

Example 1: Present Value with Monthly Compounding

Problem Statement: A Canadian pension fund manager must make a lump-sum payment of CAD 5,000,000 in 10 years. She wishes to invest in a Guaranteed Investment Contract (GIC) that pays 6% per annum, compounded monthly. What amount must be invested today?

Analysis:

  • Future Value (FVN) = CAD 5,000,000
  • Quoted annual rate (Rs) = 6% = 0.06
  • Compounding periods per year (m) = 12
  • Periodic rate (Rs/m) = 0.06/12 = 0.005
  • Number of years (N) = 10
  • Total compounding periods (mN) = 12 × 10 = 120

Solution:

PV = CAD 5,000,000 × (1.005)-120
PV = CAD 5,000,000 × 0.549633
PV = CAD 2,748,163.67

Interpretation: By investing CAD 2,748,163.67 today at 6% per annum with monthly compounding, the fund will accumulate to the required CAD 5,000,000 in 10 years.

3. Annualization of Returns: General Framework

The fundamental formula for annualizing returns from any period is based on the compound interest principle.

General Annualization Formula

Rannual = (1 + Rperiod)c - 1

Where:

  • Rannual = Annualized return
  • Rperiod = Return for the specific period
  • c = Number of periods in one year

Common Values of c:

Period Value of c Calculation Basis
Daily 365 or 250 Calendar days or trading days
Weekly 52 52 weeks per year
Monthly 12 12 months per year
Quarterly 4 4 quarters per year

4. Multi-Period Examples with Detailed Analysis

Example 2: Comparing Securities with Different Holding Periods

Problem Statement: An analyst evaluates three securities with different holding periods:

  • Security A: 6.2% return over 100 days
  • Security B: 2% return over 4 weeks
  • Security C: 5% return over 3 months

Determine which security generated the highest annualized return.

Analysis:

Security A (100-day holding period):

  • Period return = 6.2% = 0.062
  • c = 365/100 = 3.65 periods per year
  • RSA = (1 + 0.062)3.65 - 1
  • RSA = (1.062)3.65 - 1 = 1.2455 - 1
  • RSA = 0.2455 = 24.55%

Security B (4-week holding period):

  • Period return = 2% = 0.02
  • c = 52/4 = 13 periods per year
  • RSB = (1 + 0.02)13 - 1
  • RSB = (1.02)13 - 1 = 1.2936 - 1
  • RSB = 0.2936 = 29.36%

Security C (3-month/quarterly holding period):

  • Period return = 5% = 0.05
  • c = 4 periods per year
  • RSC = (1 + 0.05)4 - 1
  • RSC = (1.05)4 - 1 = 1.2155 - 1
  • RSC = 0.2155 = 21.55%

Conclusion: Security B generated the highest annualized return at 29.36%, despite having the lowest nominal period return. This demonstrates the impact of shorter holding periods when annualized.

Example 3: Exchange-Traded Fund Performance Analysis

Problem Statement: An investor evaluates three recently formed ETFs with the following performance data:

ETF Time Since Inception Return Since Inception
ETF 1 146 days 4.61%
ETF 2 5 weeks 1.10%
ETF 3 15 months 14.35%

Required: Determine which ETF has the highest annualized rate of return.

Solution:

ETF 1:

Rannual = (1.0461)365/146 - 1 = (1.0461)2.5 - 1 = 0.1193 = 11.93%

ETF 2:

Rannual = (1.0110)52/5 - 1 = (1.0110)10.4 - 1 = 0.1205 = 12.05%

ETF 3:

Rannual = (1.1435)12/15 - 1 = (1.1435)0.8 - 1 = 0.1132 = 11.32%

Answer: B (ETF 2)

Interpretation: Despite having the lowest periodic return (1.10%), ETF 2 exhibits the highest annualized return (12.05%) due to the compounding effect over multiple periods. This illustrates the critical role of the reinvestment assumption inherent in annualized return calculations.

Critical Limitation: The Reinvestment Assumption

The fundamental limitation of annualized returns is the implicit assumption that the periodic return can be precisely replicated throughout the entire year with continuous reinvestment at the same rate.

Implications:

  • Volatility is ignored in the annualization calculation
  • Market conditions may not permit consistent return replication
  • Shorter periods produce more extreme annualized figures
  • Use annualized returns for comparison, not for forecasting

5. Conversion Between Non-Annual Periods

Returns can be converted between any two time periods using the same compound interest framework.

Cross-Period Conversion Formulas

Daily to Weekly Conversion:

Rweekly = (1 + Rdaily)5 - 1

Assumes 5 trading days per week

Annual to Weekly Conversion:

Rweekly = (1 + Rannual)1/52 - 1

52 weeks comprise one year

6. Continuously Compounded Returns

Definition and Rationale

Continuously compounded return represents the rate of return under the assumption of instantaneous, continuous compounding. It is expressed as the natural logarithm of the price relative.

Key Advantage: Continuously compounded returns are additive across time periods, whereas discrete returns are multiplicative. This property simplifies mathematical modeling and statistical analysis.

Continuously Compounded Return Formula

rt,t+1 = ln(Pt+1 / Pt) = ln(1 + Rt,t+1)

Where:

  • rt,t+1 = Continuously compounded return from time t to t+1
  • ln = Natural logarithm (base e)
  • Pt+1 = Price at time t+1
  • Pt = Price at time t
  • Rt,t+1 = Discrete (holding period) return from t to t+1

Example 4: Computing Continuously Compounded Returns

Problem Statement: An asset is purchased at time t = 0 for USD 30.00. At time t = 1, the asset value is USD 34.50. Calculate both the discrete holding period return and the continuously compounded return.

Solution:

Discrete Holding Period Return:

R0,1 = (P1 - P0) / P0
R0,1 = (34.50 - 30.00) / 30.00 = 4.50 / 30.00
R0,1 = 0.15 = 15%

Continuously Compounded Return:

r0,1 = ln(P1 / P0) = ln(34.50 / 30.00)
r0,1 = ln(1.15)
r0,1 = 0.139762 = 13.98%

Observation: The continuously compounded return (13.98%) is smaller than the discrete return (15%), which is always the case for positive returns.

Verification:

P1 = P0 × exp(r0,1)
P1 = 30.00 × exp(0.139762) = 30.00 × 1.15
P1 = 34.50 ✓

7. Multi-Period Continuously Compounded Returns

Additivity Property of Continuously Compounded Returns

For an investment horizon from t = 0 to t = T:

r0,T = ln(PT / P0)

This can be decomposed as the sum of single-period continuously compounded returns:

r0,T = rT-1,T + rT-2,T-1 + ... + r0,1

Mathematical Proof:

Since PT/P0 = (PT/PT-1)(PT-1/PT-2)...(P1/P0), taking natural logarithms of both sides:

ln(PT/P0) = ln[(PT/PT-1)(PT-1/PT-2)...(P1/P0)]
= ln(PT/PT-1) + ln(PT-1/PT-2) + ... + ln(P1/P0)

Practical Applications of Continuously Compounded Returns

  • Option Pricing Models: Black-Scholes-Merton model requires continuously compounded risk-free rate
  • Quantitative Analysis: Log returns are approximately normally distributed, facilitating statistical modeling
  • Time Series Analysis: Additive property simplifies regression and forecasting models
  • Portfolio Theory: Simplified calculation of portfolio returns when combining assets

9. Exam-Relevant Nuances and Edge Cases

Critical Points for CFA Examination

  1. Day Count Convention: Be aware whether 365 (calendar days) or 250 (trading days) is specified in the question for daily return annualization.
  2. Non-Standard Periods: For periods like "100 days" or "15 months," carefully compute c as a fraction (365/100 or 12/15).
  3. Negative Returns: The annualization formula works for both positive and negative returns, but extreme negative returns can produce counterintuitive results.
  4. Consistency Requirement: All returns in a comparison must be annualized using the same methodology and day count convention.
  5. Continuously Compounded Returns: Always smaller than their discrete equivalents for positive returns; use natural logarithm (ln), not log base 10.
  6. Practical Limitation: Short-period returns, when annualized, assume perfect replication of returns—an unrealistic assumption in volatile markets.

Example 5: Comprehensive Multi-Part Problem

Problem Statement: A portfolio manager tracks the following monthly returns for a fixed income portfolio over three months:

  • Month 1: +2.5%
  • Month 2: -1.2%
  • Month 3: +3.1%

Required:

  1. Calculate the total return over the three-month period using discrete compounding
  2. Calculate the annualized return using discrete compounding
  3. Calculate the continuously compounded return for each month
  4. Calculate the total continuously compounded return and verify it equals the sum of monthly continuous returns
  5. Compare the discrete and continuous annualized returns

Solutions:

1. Total Discrete Return (3-month period):

Total Return = (1.025)(0.988)(1.031) - 1
= 1.0442 - 1
= 0.0442 = 4.42%

2. Annualized Discrete Return:

Rannual = (1 + 0.0442)4 - 1
= (1.0442)4 - 1
= 1.1871 - 1
= 0.1871 = 18.71%

3. Continuously Compounded Monthly Returns:

Month 1: r1 = ln(1.025) = 0.02469 = 2.469%
Month 2: r2 = ln(0.988) = -0.01208 = -1.208%
Month 3: r3 = ln(1.031) = 0.03053 = 3.053%

4. Total Continuously Compounded Return:

rtotal = r1 + r2 + r3
= 0.02469 + (-0.01208) + 0.03053
= 0.04314 = 4.314%

Verification: rtotal = ln(1.0442) = 0.04314 ✓

5. Annualized Continuously Compounded Return:

rannual = rquarterly × 4
= 0.04314 × 4
= 0.17256 = 17.26%

Comparison:

  • Discrete annualized return: 18.71%
  • Continuous annualized return: 17.26%
  • Difference: 1.45 percentage points
  • As expected, the continuous return is smaller
