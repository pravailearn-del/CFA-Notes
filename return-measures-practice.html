Question 1 Easy
Which return measure is most appropriate for comparing the investment skill of two portfolio managers?
A
Gross return
B
Net return
C
After-tax return

‚úì Correct Answer: A (Gross return)

Explanation:

Gross return is the most appropriate measure for comparing investment managers' skill because it reflects portfolio performance before deducting management fees and administrative expenses.

Why gross return?

  • Isolates the manager's investment decisions from fee structure
  • Larger funds typically have lower expense ratios due to economies of scale
  • Different investors may pay different fee rates for the same fund
  • Provides an apples-to-apples comparison of pure investment skill

Why not net return? Net return includes fees, which vary by fund size and investor. A skilled manager of a small fund might show lower net returns simply due to higher expense ratios, not inferior skill.

Why not after-tax return? Tax situations vary by investor (marginal rates, account types, jurisdictions), making after-tax returns inappropriate for manager comparison.

Question 2 Easy
A mutual fund reports a 12% return for the year. This figure most likely represents the fund's:
A
Gross return
B
Net return (pre-tax nominal return)
C
After-tax real return

‚úì Correct Answer: B (Net return / pre-tax nominal return)

Explanation:

When a mutual fund reports its return figure, this is the net return, which is also the pre-tax nominal return. This is what investors actually receive in their accounts after all fund expenses are deducted but before paying personal taxes.

Key Points:

  • All fund expenses and management fees have been deducted
  • No adjustment has been made for investor-specific taxes
  • No adjustment has been made for inflation
  • This is the standard reporting convention for mutual funds

Why not gross return? Funds report net returns to investors because that's what they receive. Gross returns are used internally for manager evaluation.

Why not after-tax real return? This would require knowing each investor's tax rate and the inflation rate, making it impractical for standard fund reporting.

Question 3 Medium
A fund manages $50 million in assets and earns a gross return of 18%. The fund charges a 2% management fee and has $400,000 in other annual expenses. What is the net return to investors?
A
15.2%
B
16.0%
C
15.6%

‚úì Correct Answer: A (15.2%)

Explanation:

We need to calculate the total expense ratio and subtract it from the gross return.

Step 1: Calculate total expenses

  • Management fee: $50,000,000 √ó 2% = $1,000,000
  • Other expenses: $400,000
  • Total expenses: $1,000,000 + $400,000 = $1,400,000

Step 2: Calculate expense ratio

Expense ratio = $1,400,000 / $50,000,000 = 0.028 = 2.8%

Step 3: Calculate net return

Net return = Gross return - Expense ratio

Net return = 18% - 2.8% = 15.2%

Interpretation: While the manager achieved an 18% gross return, investors received only 15.2% after the 2.8% in total expenses. This 2.8% expense ratio is relatively high and significantly impacts investor returns.

Question 4 Medium

Formula:

RAT = RPT √ó (1 - t)

Calculation:

RAT = 14% √ó (1 - 0.30)

Question 5 Medium
An investor earned a 7% nominal return in a year when inflation was 3%. The real return is closest to:
A
3.88%
B
4.00%
C
4.12%

‚úì Correct Answer: A (3.88%)

Explanation:

Real return is calculated using the Fisher equation, not by simple subtraction.

Formula:

Real Return = [(1 + Nominal Return) / (1 + Inflation)] - 1

Calculation:

Real Return = [(1 + 0.07) / (1 + 0.03)] - 1

Real Return = (1.07 / 1.03) - 1

Real Return = 1.0388 - 1

Real Return = 0.0388 = 3.88%

Why not B (4%)? This would be the linear approximation (7% - 3% = 4%), which is close but not exact. For CFA exam calculations, always use the exact formula.

Key Point: The real return (3.88%) represents the true increase in purchasing power. While the investor's account grew by 7%, prices also rose by 3%, so the actual gain in what can be purchased is only 3.88%.

Question 6 Hard
An analyst observes the following: Equities returned 10%, Treasury bills returned 3%, and inflation was 2.5%. The real return on equities and the equity risk premium are, respectively, closest to:
A
Real return: 7.32%; Risk premium: 6.80%
B
Real return: 7.50%; Risk premium: 7.00%
C
Real return: 7.32%; Risk premium: 7.00%

‚úì Correct Answer: A (Real return: 7.32%; Risk premium: 6.80%)

Explanation:

This question requires two separate calculations using the exact formula method.

Part 1: Real Return on Equities

Real return adjusts the nominal return for inflation:

Real Return = [(1 + 0.10) / (1 + 0.025)] - 1
= (1.10 / 1.025) - 1
= 1.0732 - 1
= 0.0732 = 7.32%

Part 2: Equity Risk Premium

Risk premium is the excess return over the risk-free rate (T-bills):

Risk Premium = [(1 + 0.10) / (1 + 0.03)] - 1
= (1.10 / 1.03) - 1
= 1.0680 - 1
= 0.0680 = 6.80%

Common Mistakes:

  • Using simple subtraction (10% - 2.5% = 7.5%) instead of the exact formula
  • Confusing which rate to use for risk premium calculation (use T-bills, not inflation)
  • Calculating risk premium as 10% - 3% = 7% (linear approximation)

Key Concepts:

  • Real return adjusts for inflation to show purchasing power gain
  • Risk premium shows excess return over the risk-free rate as compensation for risk
  • Always use exact formulas for CFA calculations, not approximations
Question 7 Easy
Which return measure would be most useful for comparing investment performance across different time periods with varying inflation rates?
A
Nominal return
B
Real return
C
Gross return

‚úì Correct Answer: B (Real return)

Explanation:

Real returns are the most appropriate measure for comparing performance across different time periods because they eliminate the confounding effect of varying inflation rates.

Why real returns?

Consider this example:

  • Period 1 (1980s): 15% nominal return, 8% inflation ‚Üí 6.48% real return
  • Period 2 (2010s): 8% nominal return, 2% inflation ‚Üí 5.88% real return

Looking at nominal returns alone (15% vs. 8%), Period 1 appears much better. But after adjusting for inflation, the real returns are actually quite similar (6.48% vs. 5.88%)!

Applications:

  • Comparing performance across decades
  • International comparisons where countries have different inflation rates
  • Retirement planning (need to know real purchasing power)
  • Long-term investment analysis

Why not nominal return? Nominal returns don't account for inflation, making them misleading when comparing across periods with different inflation environments.

Why not gross return? Gross return relates to fee structure, not inflation adjustment.

Question 9 Medium
If a portfolio earns an 8% return but is financed 40% with debt at a 10% interest rate, the leveraged return on equity will be:
A
Greater than 8%
B
Equal to 8%
C
Less than 8%

‚úì Correct Answer: C (Less than 8%)

Explanation:

This question tests understanding of when leverage helps versus hurts returns.

The Key Rule:

  • If RP > rD: Leverage increases return
  • If RP < rD: Leverage decreases return
  • If RP = rD: Leverage has no effect

In this case:

  • Portfolio return (RP) = 8%
  • Cost of debt (rD) = 10%
  • Since 8% < 10%, leverage will decrease the return on equity

Numerical verification:

Assuming $100 total portfolio, $40 debt, $60 equity:

  • Portfolio gain: $100 √ó 8% = $8
  • Interest cost: $40 √ó 10% = $4
  • Net to equity: $8 - $4 = $4
  • Return on equity: $4 / $60 = 6.67%

Result: The 8% portfolio return becomes 6.67% on equity - worse than unleveraged!

Why? You're borrowing at 10% to invest in something earning only 8%. You're losing 2% on every dollar borrowed. This is like taking out a credit card cash advance at 20% to put in a savings account earning 2% - you'd lose money!

Key Lesson: Leverage only benefits you when your investment return exceeds your borrowing cost. Otherwise, it amplifies your losses.

Question 11 Medium

This question illustrates the economies of scale advantage for large funds.

  • Expense ratio: $500,000 / $20,000,000 = 0.025 = 2.5%

  • Assets: $2,000 million
  • Fixed expenses: $500,000
  • Expense ratio: $500,000 / $2,000,000,000 = 0.00025 = 0.025%

14.975% - 12.5% = 2.475 percentage points ‚âà 2.48 pp

Key Insights:

  • Both managers have identical investment skill (15% gross return)
  • The same $500,000 in expenses represents 2.5% for the small fund but only 0.025% for the large fund
  • The large fund's investors receive 2.48 percentage points more return solely due to economies of scale
  • This is why small funds often waive part of their expenses to remain competitive
