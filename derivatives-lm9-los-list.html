Module Overview: This module explores put-call parity, a fundamental relationship linking European calls, puts, the underlying asset, and risk-free bonds. You'll learn how to use put-call parity to replicate option positions, identify arbitrage opportunities, and extend these concepts to include forward contracts through put-call forward parity.

Explain put‚Äìcall parity for European options
Master the put-call parity relationship for European options: C + X/(1+r)^T = P + S, where C is call price, P is put price, S is current stock price, X is strike price, r is risk-free rate, and T is time to expiration. This equation states that a portfolio of a long call plus a risk-free bond (paying X at maturity) equals a portfolio of a long put plus the underlying stock. Both portfolios have identical payoffs at expiration. Use put-call parity to: replicate any position using the other three instruments, identify arbitrage opportunities when the relationship is violated, and calculate implied prices when three of the four values are known. Rearrange to create synthetic positions: synthetic call (C = P + S - PV(X)), synthetic put (P = C - S + PV(X)), synthetic stock (S = C - P + PV(X)), synthetic bond (PV(X) = P - C + S).

Explain put‚Äìcall forward parity for European options
Understand put-call forward parity, which incorporates forward contracts: C + PV(F) = P + PV(X), or equivalently C - P = PV(F - X), where F is the forward price for delivery at option expiration. This relationship shows that the difference between call and put prices equals the present value of the difference between the forward price and strike price. Put-call forward parity is useful when dealing with options on futures or when forward contracts are more liquid than the spot market. It allows synthetic replication using forwards instead of spot positions: a synthetic call equals a long put plus a long forward contract; a synthetic put equals a long call plus a short forward contract. This relationship must hold to prevent arbitrage opportunities.
