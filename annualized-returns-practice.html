Question 1 Easy
An investor earns a return of 2% over one month. What is the annualized return?
A
24.0%
B
26.8%
C
22.5%

‚úì Correct Answer: B (26.8%)

Explanation:

To annualize a monthly return, we use the formula: Rannual = (1 + Rmonthly)12 - 1

Here:

  • Rmonthly = 2% = 0.02
  • c = 12 (there are 12 months in a year)

Calculation:

Rannual = (1 + 0.02)12 - 1

Rannual = (1.02)12 - 1

Rannual = 1.2682 - 1 = 0.2682 = 26.8%

Why not A? Option A (24%) is what you'd get if you simply multiplied 2% √ó 12, but this ignores the compounding effect. The correct answer accounts for the fact that each month's return compounds on the previous months' returns.

Question 2 Easy
Which of the following statements about continuously compounded returns is most accurate?
A
Continuously compounded returns are always larger than discrete returns for positive price changes.
B
Continuously compounded returns can be added across time periods.
C
Continuously compounded returns are preferred for performance reporting to clients.

‚úì Correct Answer: B

Explanation:

The key advantage of continuously compounded returns is their additivity property. For multiple periods, you can simply add the continuously compounded returns:

rtotal = r1 + r2 + r3 + ...

In contrast, discrete returns must be multiplied: (1 + R1)(1 + R2)(1 + R3) - 1

Why not A? This is incorrect. Continuously compounded returns are always smaller than discrete returns for positive price changes. For example, if a stock rises from $100 to $115 (15% discrete return), the continuous return is ln(1.15) = 13.98%.

Why not C? While continuous returns have mathematical advantages, discrete returns are actually preferred for client reporting because they're more intuitive. Clients understand "I made 15%" better than "I made 13.98% continuously compounded."

Question 3 Medium
A stock returned 3.5% over a 45-day holding period. The annualized return, assuming 365 days per year, is closest to:
A
28.4%
B
31.2%
C
33.9%

‚úì Correct Answer: C (33.9%)

Explanation:

For non-standard periods like 45 days, we calculate c as the number of 45-day periods in a year:

c = 365 / 45 = 8.111

Calculation:

Rannual = (1 + 0.035)8.111 - 1

Rannual = (1.035)8.111 - 1

Rannual = 1.339 - 1 = 0.339 = 33.9%

Key Insight: When the holding period doesn't fit evenly into a year (like 45 days), you must use the fractional exponent. Don't try to simplify to quarterly or monthly periods when the question specifies an exact number of days.

Question 4 Medium
An investment generated a total return of 18% over a 15-month period. The annualized return is closest to:
A
14.4%
B
15.2%
C
22.5%

‚úì Correct Answer: A (14.4%)

Explanation:

When the holding period is longer than one year, we need to use a fractional exponent less than 1. One year is 12 months out of a 15-month period, so:

c = 12 / 15 = 0.8

Calculation:

Rannual = (1 + 0.18)0.8 - 1

Rannual = (1.18)0.8 - 1

Rannual = 1.144 - 1 = 0.144 = 14.4%

Logic Check: The annualized return (14.4%) is lower than the total return (18%) because the 18% was earned over more than one year (15 months). This makes sense - you'd need to earn 14.4% each year for 1.25 years to end up with 18% total.

Why not C? Option C would suggest you're earning more on an annual basis than you did over 15 months, which is impossible without additional returns.

Question 5 Medium
A security's price increased from ‚Çπ1,200 to ‚Çπ1,380. The continuously compounded return is closest to:
A
13.98%
B
15.00%
C
16.25%

‚úì Correct Answer: A (13.98%)

Explanation:

To calculate continuously compounded return, we use the natural logarithm of the price relative:

r = ln(Pt+1 / Pt)

Calculation:

r = ln(1,380 / 1,200)

r = ln(1.15)

r = 0.1398 = 13.98%

Comparison: The discrete return would be (1,380 - 1,200) / 1,200 = 15%. Notice that the continuously compounded return (13.98%) is slightly smaller than the discrete return (15%), which is always the case for positive returns.

Calculator Tip: Make sure you're using the "ln" button (natural log), not "log" (base-10 logarithm)!

Which ETF has the highest annualized return?

Explanation:

We need to calculate the annualized return for each ETF:

c = 365 / 73 = 5.0

Rannual = (1.028)5.0 - 1 = 1.1475 - 1 = 14.75%

c = 52 / 6 = 8.667

Rannual = (1.015)8.667 - 1 = 1.1387 - 1 = 13.87%

Question 7 Easy
Which of the following is the primary advantage of using continuously compounded returns over discrete returns?
A
They are easier for clients to understand.
B
They are additive across time periods.
C
They always produce larger return figures.

‚úì Correct Answer: B

Explanation:

The main mathematical advantage of continuously compounded returns is their additivity. If you have returns for Period 1, 2, and 3:

  • Discrete returns: Total = (1 + R1)(1 + R2)(1 + R3) - 1 (multiplication)
  • Continuous returns: Total = r1 + r2 + r3 (simple addition)

This property makes continuous returns much easier to work with in statistical models, time series analysis, and option pricing formulas like Black-Scholes.

Why not A? Continuously compounded returns are actually less intuitive for clients. Telling a client "you made 13.98% continuously compounded" is less clear than "you made 15%."

Why not C? This is backwards. Continuously compounded returns are always smaller than discrete returns for positive price changes.

Question 8 Hard
A portfolio experiences the following monthly returns: +3.2%, -1.8%, +2.5%, +1.1%. The annualized return using discrete compounding is closest to:
A
15.2%
B
16.4%
C
17.1%

‚úì Correct Answer: B (16.4%)

Explanation:

This is a multi-step problem. First, we need to find the total return over the 4-month period, then annualize it.

Step 1: Calculate 4-month total return

Total = (1.032)(0.982)(1.025)(1.011) - 1

Total = 1.0512 - 1 = 0.0512 = 5.12%

Step 2: Annualize the 4-month return

Since there are 3 four-month periods in a year (12 months / 4 months = 3):

Rannual = (1.0512)3 - 1

Rannual = 1.1639 - 1 = 0.1639 = 16.4%

Key Points:

  • With multiple period returns, always compound them first to get the total period return
  • Notice we kept the negative return as 0.982 (= 1 - 0.018) when multiplying
  • The order of multiplication doesn't matter: (1.032)(0.982)(1.025)(1.011) gives the same result regardless of order
Question 9 Medium
An analyst observes the following: A stock's price was $50 at the beginning of the year and $57.50 at year-end. What is the difference between the discrete annual return and the continuously compounded annual return?
A
0.98 percentage points
B
1.02 percentage points
C
1.15 percentage points

‚úì Correct Answer: B (1.02 percentage points)

Explanation:

Discrete Annual Return:

R = (57.50 - 50.00) / 50.00 = 7.50 / 50.00 = 0.15 = 15.00%

Continuously Compounded Annual Return:

r = ln(57.50 / 50.00) = ln(1.15) = 0.1398 = 13.98%

Difference:

15.00% - 13.98% = 1.02 percentage points

Important Terminology Note: When we subtract percentages, we express the difference as "percentage points," not "percent." If something goes from 10% to 12%, it increased by 2 percentage points, not by 2% (which would mean 10% √ó 1.02 = 10.2%).

Pattern to Remember: For typical investment returns (say, in the range of -20% to +30%), the difference between discrete and continuous returns is usually 0.5 to 1.5 percentage points, with larger discrete returns showing bigger differences.

Question 10 Hard
A fund manager needs to invest CAD 3,000,000 today to have CAD 5,000,000 in 8 years. A Guaranteed Investment Contract (GIC) offers 6.5% per annum with quarterly compounding. Will this investment meet the manager's goal?
A
Yes, it will grow to approximately CAD 5,088,000
B
No, it will only grow to approximately CAD 4,876,000
C
Yes, it will grow to approximately CAD 5,250,000

‚úì Correct Answer: A (Yes, approximately CAD 5,088,000)

Explanation:

We need to calculate the future value of CAD 3,000,000 invested for 8 years at 6.5% with quarterly compounding.

Given:

  • PV = CAD 3,000,000
  • Rs = 6.5% = 0.065 (quoted annual rate)
  • m = 4 (quarterly compounding)
  • N = 8 years
  • Periodic rate = Rs/m = 0.065/4 = 0.01625
  • Total periods = mN = 4 √ó 8 = 32

Formula:

FV = PV √ó (1 + Rs/m)mN

Calculation:

FV = CAD 3,000,000 √ó (1.01625)32

FV = CAD 3,000,000 √ó 1.6961

FV = CAD 5,088,300

Conclusion: Yes, the investment will meet the goal. The CAD 3,000,000 investment will grow to approximately CAD 5,088,000, which exceeds the required CAD 5,000,000 by about CAD 88,000.

Note on Compounding Frequency: Quarterly compounding means the interest is calculated and added to the principal four times per year, which results in more growth than annual compounding would provide.

Question 11 Medium
An investor earned the following continuously compounded monthly returns: 2.2%, 1.8%, -0.5%, 2.1%. What is the total continuously compounded return for the four-month period?
A
5.6%
B
5.9%
C
6.2%

‚úì Correct Answer: A (5.6%)

Explanation:

This question tests the key property of continuously compounded returns: they are additive!

Calculation:

rtotal = r1 + r2 + r3 + r4

rtotal = 2.2% + 1.8% + (-0.5%) + 2.1%

rtotal = 2.2% + 1.8% - 0.5% + 2.1%

rtotal = 5.6%

Why This Works: The beauty of continuously compounded returns is that you can simply add them together to get the total return. No multiplication needed!

Comparison with Discrete Returns: If these were discrete returns instead of continuous returns, you would have had to calculate:

(1.022)(1.018)(0.995)(1.021) - 1 = 5.76%

Notice that:

  • Continuous total: 5.6%
  • Discrete total: 5.76%
  • The continuous return is slightly smaller, as expected

Practical Tip: In quantitative finance and statistical modeling, this additivity property is why continuously compounded returns are preferred. They make the math much simpler!

Question 12 Hard
An analyst is comparing two bonds:
  • Bond A: Offers 7.2% per annum with semiannual compounding
  • Bond B: Offers 7.1% per annum with monthly compounding
If an investor has ‚Çπ1,000,000 to invest for 5 years, how much more will the better bond provide?
A
Bond A will provide approximately ‚Çπ2,700 more
B
Bond B will provide approximately ‚Çπ1,500 more
C
Bond A will provide approximately ‚Çπ5,200 more

‚úì Correct Answer: B (Bond B will provide approximately ‚Çπ1,500 more)

Explanation:

We need to calculate the future value for both bonds and compare.

Bond A (7.2% semiannual):

  • Rs = 0.072
  • m = 2 (semiannual)
  • Periodic rate = 0.072/2 = 0.036
  • Total periods = 2 √ó 5 = 10

FVA = ‚Çπ1,000,000 √ó (1.036)10

FVA = ‚Çπ1,000,000 √ó 1.4214

FVA = ‚Çπ1,421,400

Bond B (7.1% monthly):

  • Rs = 0.071
  • m = 12 (monthly)
  • Periodic rate = 0.071/12 = 0.005917
  • Total periods = 12 √ó 5 = 60

FVB = ‚Çπ1,000,000 √ó (1.005917)60

FVB = ‚Çπ1,000,000 √ó 1.4229

FVB = ‚Çπ1,422,900

Difference:

‚Çπ1,422,900 - ‚Çπ1,421,400 = ‚Çπ1,500

Key Insight: Even though Bond A has a higher stated rate (7.2% vs 7.1%), Bond B provides more value because it compounds more frequently (monthly vs semiannually). The more frequent compounding compensates for the lower stated rate.

General Principle: When comparing investments with different compounding frequencies, you cannot simply compare the stated rates. The effective annual rate is what matters, and more frequent compounding increases the effective rate.

Effective Annual Rates:

  • Bond A: (1.036)2 - 1 = 7.33%
  • Bond B: (1.005917)12 - 1 = 7.34%

Bond B's effective rate is slightly higher, hence the better outcome.
